# 🚀 NextGen Founders

NextGen Founders is a platform that helps young people in Latin America (ages 14–18) learn how to build their first business while they are still in school.

## 🌍 What it does

- Teaches entrepreneurship step by step  
- Helps users build real businesses (not just theory)  
- Includes an AI mentor for guidance  
- Allows users to document their progress (founder log)  
- Connects a community of young founders  

## 🧠 Features (MVP)

- 🤖 AI Mentor (Gemini API)
- 📓 Founder Log (coming soon)
- 📊 Progress tracking (coming soon)
- 💬 Community discussion (coming soon)
- 👑 Admin panel (coming soon)

## ⚙️ Tech Stack

- Node.js + Express
- AI: Google Gemini API
- Deployment: Railway
